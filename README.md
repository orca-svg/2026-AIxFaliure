## 이 저장소에 대하여

이 저장소는 제안서를 작성한 협업 과정(리서치, 초안, 리뷰)을 담고 있습니다.
- 제안서 소스: [`proposal/proposal.typ`](proposal/proposal.typ)
- 리서치·근거: [`research/`](research/)
- 팀 회의 요약: [`research/meetings/`](research/meetings/) — 회의별 논의·결정 사항 정리본
- 프로토타입: [`tool/` 언노운 나침반](tool/) — 대응 방안(AI 리터러시)을 체험할 수 있는 4분면 자가 진단 도구
- 확정 PDF: [Releases](../../releases)

전체 제출작은 KAIST 실패연구소의 AI 인식 관련 연구 자료로 익명 활용될 수 있습니다.
