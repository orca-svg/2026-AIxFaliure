1 AI에게 시키려는 작업을 설명하세요

무엇을 만들려는지, 이 분야·코드베이스에 얼마나 익숙한지 솔직하게 적을수록 진단이 정확해집니다.

2 진단 프롬프트를 복사해 쓰는 AI에게 붙여넣으세요

ChatGPT, Claude, Gemini — 어디든 좋습니다. API 키 없이, 프롬프트 자체가 도구입니다.
