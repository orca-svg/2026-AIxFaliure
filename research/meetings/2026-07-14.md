# 팀 회의 요약 — 2026-07-14

**형식**: 취합 회의 — 1번 내러티브 리뷰 + 3번 방향 재정렬 + 제출 형식 확정 (주요 결정 다수)

## 1. 1번(예견된 실패) 내러티브 "최적화된 아이들" 리뷰

- 팀원이 작성한 15페이지 서사 + 연도별 로드맵 표 공유 (7/10 결정대로 디스토피아적 디테일 서사).

| 연도 | 사건 |
|---|---|
| 2026 | AI의 시험 도입 여부·허용 평가 논쟁 |
| 2027 | AI 튜터 사업 본격화, 학생 맞춤형 보급 시작 |
| 2028 | 전력 그리드 병목 + 메모리(삼성·하이닉스) 수급 — "전력 부족"은 예측 가능한 사실 기반 |
| 2029 | 중국 로봇 건설 셀(SMR) 급진 런칭·테슬라 대응 → AI로 에너지 건설기간 압축, 전력난 극복. 프로젝트 단위 실적 = 국가경쟁력 |
| 2030 | "시험 아닌 프로젝트 단위 결과물" 컨센서스 공고화 |
| 2031 | 공교육 AI 튜터 90%+ 보급(1인당 2~3개), 교사 다수 이탈 |
| 2032 | A2A 멀티에이전트 시대 — 빅테크 chaperone 레이어 도입하나 에이전트·유저 모두 우회 |
| 2033~ | AI 완전 의탁 — "문제는 잘 푸는데 왜 푸는지 모름", 성과만 남고 방향 상실 |

- **서사의 핵심 설계**: 세대 간 갈등이 아니라 **모두가 선의로 행동했는데 메타인지 부재로 실패**하는 구조.
  악의적 AI 사용자는 한 명도 등장하지 않음. 교사는 AI를 무비판적으로 수용하고 "개입하지 않는"
  인물로 의도적으로 설계 — 학생을 평가하는 매트릭은 프로젝트 성과뿐이라 학생은 효율만 좇고,
  교사는 "고민 과정이 곧 성장"이라 믿는 **미스매치**가 실패의 동력.
- 로드맵 일부가 실제 발표된 다개년 계획과 우연히 일치함을 확인(2029 등) — 표는 원본 유지.

## 2. 3번(대응 방안) 논쟁 — "AI 리터러시가 답인가?"

- 서로 독립적으로 같은 의문 제기: AI 리터러시 서술만으로 앞의 실패를 풀 수 있는가.
- **합의된 재해석**: 3번의 메타인지 회고(2x2 Known-Unknown Matrix)를 1번 시나리오에
  역대입하면 2036년 실패가 방지된다 — 이 관계를 드러내는 것이 논지의 핵심.
  1번은 "3번이 없는 세계"의 사고실험으로 기능한다.
- **확장**: 언노운 나침반은 학생뿐 아니라 **교수자에게도** 필요 — 대시보드 형태(학생·학부모·교사가
  이해도 회고를 파악)로 보편 도입하거나, AI 튜터 워크플로에 회고 스텝을 자연스럽게 내장하는 방향.
- 핵심 위기 문장은 볼드나  한 워딩으로 재작성 — 이 관점은 의도적으로 봐야만 보이므로
  평가자에게 명시적으로 드러낼 것.

## 3. 원인 진단(2번) 재프레이밍 + 실패 축 조정

- **원인 재프레이밍**: "AI 때문에 인간 평가가 무의미해졌다"(→ "인간은 지적 능력이 필요 없다"로
  오독될 위험, 하위의 인지적 비계 논지와 충돌) 대신 **"AI 성과 효용이 커지며 사회가 인간의
  지적 능력을 경시하는 풍조가 형성된 것"**을 핵심 원인으로. 해당 문단은 이전 버전으로 revert.
- **실패 축 4가지 중 ③(AI 교육자의 악영향/인성교육 공백) 제외** — 맥락상 가장 이탈,
  ①②④에 집중. 3번의 "대상적 확장" 비중 과다도 함께 해소되어 논리 구조 유지.
- **양극화(축 ②)는 메인에서 제외** — 최근 모델 접근 제한 이슈로 다수 지원자가 같은 주제로
  제출할 것으로 예상(참신성↓). 필요 시 2번의 증거 레퍼런스로만 활용(유학=수출품 앵글).
- **레퍼런스는 2번에 집중 배치** — 3번에 추가 인용은 어색. 에이전티즘 학회 세션
  (외주화→사고과정 생략→과정 평가 필요)은 1번 서사에도 접목 가능.
- 미확정: 지자체 인수위 교육 공약 인용 여부 — 공약이 백서 등으로 공개됐는지 확인 후 결정

## 4. 제출 형식 확정

- 심사 기준 재확인: 배점이 **"예리한 실패 지점 포착 + 논리/접근"에 집중(≈70점)** →
  무게중심을 솔루션이 아니라 문제 정의·원인 진단에.
- **제출 = 최대 2쪽 시놉시스(문어체)** + GitHub 레포를 추가 자료 링크로. 콘티/영상화는 시간상 보류.
- 1번 서사는 결말 변경 없이 유지(메타인지가 들어간 대안 결말은 3번의 역할을 지워버림) —
  공모전 채점 기준에 맞춰 압축만.
- 2번은 **표 형식**으로 작성(제목+줄글 형식은 2쪽 제약에서 가독성 낮음).
- 전체 내러티브·회의록·도구는 이 레포에 정리해 제출.
